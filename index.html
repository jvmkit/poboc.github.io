本脚本适用环境

系统支持：CentOS 6+，Debian 7+，Ubuntu 12+
内存要求：≥128M
日期　　：2017 年 02 月 13 日

关于本脚本

1、一键安装 Shadowsocks-Python， ShadowsocksR， Shadowsocks-Go， Shadowsocks-libev 版（四选一）服务端；
2、各版本的启动脚本及配置文件名不再重合；
3、每次运行可安装一种版本；
4、支持以多次运行来安装多个版本，且各个版本可以共存（注意端口号需设成不同）；
5、若已安装多个版本，则卸载时也需多次运行（每次卸载一种）；
6、Shadowsocks-Python 和 ShadowsocksR 安装后不可同时启动（因为本质上都属 Python 版）。

概述

宗旨

Markdown 的目标是实现「易读易写」。

可读性，无论如何，都是最重要的。一份使用 Markdown 格式撰写的文件应该可以直接以纯文本发布，并且看起来不会像是由许多标签或是格式指令所构成。Markdown 语法受到一些既有 text-to-HTML 格式的影响，包括 Setext、atx、Textile、reStructuredText、Grutatext 和 EtText，而最大灵感来源其实是纯文本电子邮件的格式。

我排着队拿着爱的号码牌

发表于 2013-11-24 | 分类于 故事 |

　在城市嘉年华上，整个会场火树银花，欢呼声振耳欲聋。
　　易乐就是在那人影绰绰里，看到六年后的简年。
　　头上戴着硕大的兔头箍，毛茸茸的耳朵耸拉着。粉红色的长毛外套，把她包得娇小，比读高中时更小了。
　　在扮演嘉年华里的人偶，随着人群起舞，跟着歌声轻和。
　　他突然情绪失控，大步踱了过去，揪住她的手臂：“你好吗?”
　　没有电影里彼此拥吻的剧情，她怔住，迷茫的表情缓缓地舒展开。最后仍是双目失焦地迎着他的睛睛：“你是?”
　　周围的欢呼声戛然而止，喧嚣这一刻沉寂下去。是搭建的临时舞台上，
　　有人已经在台上唱起了另一首歌：“我往前飞飞过一片时间海，我们也常在爱情里受伤害。”
　　她轻轻地跟着合了起来，依然是六年前那般空灵的嗓音。他一直都记得，她曾在阳台上晾晒衣物时唱过这首歌。
　　那时候，他们青春正好，而如今歌声依旧，人事全非。
